# geometric_lib

### Документация

Данный документ является документацией к репозиторию ```geometric_lib``` и содержит
- Общее описание проекта
- Описание каждой функции с примерами вызова

## Общее описание
В данном репозитории собраны файлы ```circle.py```, ```square.py```, ```triangle.py``` и ```rectangle.py```, написанные на языке ```python``` и содержащие функции для вычисления площади и периметра для круга, квадрата, треугольника и прямоугольника.

### Использованные библиотеки
В проекте используется встроенный модуль ```math```

## _```circle.py```_
---
Содержит функции для вычисления площади и периметра (длины окружности) круга.
#### Код
```python
import math

def area(r):   
    return math.pi * r * r

def perimeter(r):
	return 2 * math.pi * r
```
---
#### <code style="color:DarkTurquoise">float</code>```area(r)```

Вычисляет площадь (_S_) круга радиусом ```r``` по формуле _S = πR²_.
##### Параметры:
- ```r (int)/(float)```: целое или вещественное положительное число, радиус круга

##### Возвращаемое значение:
- ```(float)```: положительное число с плавающей точкой, площадь круга радиусом ```r```


##### Пример вызова:
```python
from circle import area

r = 4
print(area(r))
```
Вывод
```
50.26548245743669
```
---
#### <code style="color:DarkTurquoise">float</code>```perimeter(r)```

Возвращает длину окружности (_P_) круга радиусом ```r``` по формуле _P = 2πR_.

##### Параметры:
- ```r (int)/(float)```: целое или вещественное положительное число, радиус круга

##### Возвращаемое значение:
- ```(float)```: положительное число с плавающей точкой, длина окружности круга радиусом ```r```

##### Пример вызова:
```python
from circle import perimeter

r = 4
print(perimeter(r))
```
Вывод
```
25.132741228718345
```

## _```triangle.py```_
---
Содержит функции для вычисления площади и периметра произвольного треугольника.
#### Код
```python
def area(a, b, c):
	p = (a + b + c)/2  
	s = (p * (p - a) * (p - b) * (p - c)) ** 0.5  
	return s

def perimeter(a, b, c):
	return a + b + c
```
---
#### <code style="color:DarkTurquoise">float</code>```area(a, b, c)```

Возвращает площадь (_S_) произвольного треугольника со сторонами длиной ```a```, ```b``` и ```c``` по формуле _S = _$\sqrt{p(p-a)(p-b)(p-c)}$.
##### Параметры:
- ```a (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата
- ```b (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата
- ```c (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата

##### Возвращаемое значение:
- ```s (float)```: положительное число с плавающей точкой, площадь заданного треугольника

##### Пример вызова:
```python
from triangle import area  
  
a, b, c = 3, 4, 5  
print(area(a, b, c))  
```
Вывод
```
6.0
```
Пример с треугольником посложнее:
```python
from triangle import area  
  
a, b, c = 6, 2, 7  
print(area(a, b, c)) 
```
Вывод
```
5.562148865321747
```

---
#### <code style="color:DarkTurquoise">int</code>```perimeter(a, b, c)```

Возвращает площадь (_S_) произвольного треугольника со сторонами длиной ```a```, ```b``` и ```c``` по формуле _S = a+b+c_.

##### Параметры:
- ```a (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата
- ```b (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата
- ```c (int)/(float)```: целое или вещественное положительное число, длина стороны квадрата

##### Возвращаемое значение:
- ```(int)/(float)```: целое или вещественное положительное число, периметр заданного треугольника
##### Пример вызова:
```python
from triangle import perimeter  
  
a, b, c = 6, 2, 7  
print(perimeter(a, b, c))
```
Вывод
```
15
```

## _```rectangle.py```_
---
Содержит функции для вычисления площади и периметра прямоугольника.
#### Код
```python
def area(a, b):
	return a * b

def perimeter(a, b):
	return 2 * (a + b)
```
---
#### <code style="color:DarkTurquoise">int</code>```area(a, b)```

Возвращает площадь (_S_) прямоугольника со смежными сторонами длиной ```a``` и ```b``` по формуле _S = ab_.
##### Параметры:
- ```a (int)/(float)```: целое или вещественное положительное число, длина одной смежной стороны
- ```b (int)/(float)```: целое или вещественное положительное число, длина второй смежной стороны

##### Возвращаемое значение:
- ```(int)/(float)```: целое или вещественное положительное число, площадь заданного прямоугольника 

##### Пример вызова:
```python
from rectangle import area  
  
a, b = 4, 7  
print(area(a, b))
```
Вывод
```
28
```

---
#### <code style="color:DarkTurquoise">int</code>```perimeter(a, b)```

Возвращает периметр (_P_) прямоугольника со смежными сторонами длиной ```a``` и ```b``` по формуле _P = 2(a+b)_.

##### Параметры:
- ```a (int)/(float)```: целое или вещественное положительное число, длина одной смежной стороны
- ```b (int)/(float)```: целое или вещественное положительное число, длина второй смежной стороны

##### Возвращаемое значение:
- ```(int)/(float)```: целое или вещественное положительное число, периметр заданного прямоугольника 

##### Пример вызова:
```python
from rectangle import perimeter  
  
a, b = 4, 7  
print(perimeter(a, b))
```
Вывод
```
22
```
